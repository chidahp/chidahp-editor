# ✥ เครื่องมือจัดหน้าแบบชี้ดาบ (Chidahp Web Formatting & Pacing Editor)

ยินดีต้อนรับสู่เว็บแอปพลิเคชันสำหรับนักเขียนและผู้จัดรูปเล่มนิยาย/หนังสือ เพื่อร่วมสัมผัสและนำหลักการจัดหน้าสไตล์ **"ชี้ดาบ" (Chidahp)** มาประยุกต์ใช้เพื่อควบคุมจังหวะการเล่าเรื่อง (Pacing) บริหารสายตา และสื่อสารอารมณ์ความรู้สึกตรงสู่หัวใจผู้อ่านอย่างมีประสิทธิภาพสูงสุด!

---

## 📖 อะไรคือการจัดหน้าแบบชี้ดาบ?
การเขียนแบบชี้ดาบคือ **"Emotional Writing" (การเขียนเพื่อสื่อสารอารมณ์)** โดยตรง โดยอาศัยศาสตร์แห่งการจัดจังหวะเพื่อลดภาระการกวาดสายตาของผู้อ่าน และเน้นประเด็นสำคัญที่ผู้เขียนต้องการเน้นย้ำ เครื่องมือนี้จะถอดแบบกฎเกณฑ์การเคาะเว้นวรรคและการตัดคำจาก "ชี้ดาบ" มาจำลองไว้ให้ใช้งานได้อย่างราบรื่น

---

## ✨ ฟีเจอร์เด่น
1. **Interactive Dual-Pane Workspace**: ห้องเขียนอักษรไร้สิ่งรบกวนฝั่งซ้าย ควบคู่กับตัวจำลองหน้าหนังสือจริงขนาด Pocket Book ฝั่งขวาแบบเปิดหน้าคู่ (Spread) ขยับเลื่อนตามแบบเรียลไทม์
2. **Smart Rhythm Linter (ตัวเตือนจังหวะการเล่า)**: แจ้งเตือนย่อหน้ายาวเกินเกณฑ์ (เกิน 3 บรรทัด), มีจุดพักซีนแช่, หรือประโยคสั้นติดกันเป็นพืด โดยมีปุ่มแก้ไขด่วน (Auto-Fix) ให้คลิกทีเดียวเปลี่ยน
3. **Quick Toolbar (แถบตัวช่วยเคาะจังหวะ)**:
   - `[H]` บรรทัด Hook: แปลงประโยคเป็นประโยคฮุกเดี่ยวที่เว้นระยะห่างด้านบน-ล่างให้สวยงามสมดุล
   - `...` จุดหยุด: แทรกช่วงหน่วงอารมณ์ในการอ่านเพื่อขยายบรรทัด
   - `✥` ตัวคั่นการอ่าน: ใส่สัญลักษณ์แบ่งตอนให้สมองผู้อ่านได้หยุดพัก
4. **Virtual Customizers (ตัวเลือกรูปเล่ม)**: ปรับขนาดอักษร, ย่อหน้า, ระยะขอบกระดาษ, ฟอนต์ หรือเปลี่ยนจำนวนบรรทัดต่อหน้าเพื่อดูทิศทางการไหลของอักษร
5. **Interactive Lessons**: พ่วงตัวอย่างแบบฝึกหัดจริงในระบบ 5 บทเรียน เพื่อฝึกตัดซีนแช่ วางประโยคฮุก และแบ่งย่อหน้าให้เป็นอย่างชี้ดาบ
